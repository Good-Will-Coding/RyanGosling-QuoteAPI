Ryan Gosling Quotes API

Hey, girl. So I heard you need an API for all your Ryan Gosling quotes.
Well, you came to the right place. If you need a Gosling quote, we've got a quote.

How to use our API

To retrieve all Ryan Gosling quotes, send a GET request to the following:

 
https://ryangosling-quotes.herokuapp.com/api/quotes

Example response for quotes request 

 [ 
    { 
      "quote": "I don't have wheels on my car.", 
      "movie": "Drive" 
    }
 ]

To retrieve a random quote, send a GET request to the following:

 
https://ryangosling-quotes.herokuapp.com/api/quotes/random

Example response for random quote 

[
    {

    "quote": "If you're a bird, I'm a bird.",
    "movie": "The Notebook"
    },
]

For a certain amount of quotes,
send a GET request and change the number parameter to the desired amount of quotes you wish to receive back:

 
https://ryangosling-quotes.herokuapp.com/api/quotes/{number}

Example Request

https://ryangosling-quotes.herokuapp.com/api/quotes/2
Example response for numbered request 

[
  {
    "quote": "Hey, you want a toothpick?",
    "movie": "Drive"
  },
  {
    "quote": "My hands are a little dirty.",
    "movie": "Drive"
  },
]
